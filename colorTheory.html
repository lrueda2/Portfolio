DES 170: Color Theory

This class is the intro to color theory, aesthetics, and systems with emphasis on the definition, replication, and interaction of color in design applications across various media. I develop sensitivity towards color through intense observation of color value. Also, learned the difference between color values by sorting/comparing hues. Improve communication skills by using color terms (hue, achromatic scale, color contrast, saturation, value, tint, shade).
